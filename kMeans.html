The animation above shows K Means Clustering where K = 3 for a dataset of 500 randomly placed points




Each hidden data cluster can be thought of as a planet with a gravitational pull. The cluster will pull the nearest centroid closer to the center of it each step of the algorithm.





The Algorithm



  • Input:

    • K (# of clusters)
    • A data set

  • Setup - Randomly generate starting positions for the K many centriods


  • Data Assignment: Assign each datapoint to the nearest centroid


  • Centroid Update: For each centroid, calculate the mean distance of all data points assigned to it, update centroid location to mean location


  • Repeat Steps 1 and 2 until centroids no longer update





In a Nutshell



Category

Unsupervised Learning- Data is not labeled

Usage:

Classification (Clustering) - Finding groups within the data


  • K Means can be thought of as clustering data into K groups.
  • After the variable K is defined, (for example purposes we will use 3) The data that is already present will separate itself into K groups.
  • Thinking of this algorithm visually, it appears as if the data actually "reveals" itself into K groups.
  • For all Clustering algorithms remember, they are searching and exposing data groups based on certain qualities within the data.
  • Over each run of the algorithm the groups become more and more defined.


Similarities in qualities of certain data points compared to others create groups

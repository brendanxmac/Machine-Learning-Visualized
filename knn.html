K Nearest Neighbors








People are placed into 3 groups based on their hair color, eye color, height. When adding another person to the group, look at that persons K nearest neighbors (K is a predetermined integer) and classify (place) the new person into the group the majority of those neighbors belonged to.







The Algorithm



  • Input:

    • K (# neighbors to check)
    • A training set & a testing data set

  • Setup - Plot the data onto the graph and show class separation


  • Find K Nearest NeighborsPlot a new data point and find it's K nearest neighbors using Euclidean distance


  • Classify SampleDetermine which class the majority of the samples K nearest neighbors belongs to, classify the sample as that majority class.





In a Nutshell



Category

Supervised Learning- Data is labeled

Usage:

Classification - Finding groups within the data


  • K Nearest Neighbors can be thought of as classifying data into K groups based on previously classified data.
  • After the variable K is defined, (for example purposes we will use 3) the new sample data points which are part of the testing data set will find the K nearest neighbors
  • Thinking of this algorithm visually, it appears as if the data which group it is more closely related to, and is classified as such


Find the training data points you are nearest to, join the majority group of those neighbors

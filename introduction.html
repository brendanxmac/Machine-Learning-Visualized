What is Machine Learning?




Machine learning is software written by a programmer, just like any other code.


It is software that 'learns': by collecting data and running an algorithm on that data..
  • Machine learning involves the development of computer programs that look at data and learn from it.
  • Four different types of machine learning exist: supervised, unsupervised, semi-supervised, and reinforcement learning.
  • Machine Learning has the ability to make correlations that humans might not be able to make on their own, as well as process and analyze huge sets of data.

Note: In machine learning, the input is data (and sometimes expected output), and the actual output is a program that learned something about the data (and sometimes expected output) that can then be fed more data to make better predictions when fed more input


Types of Machine Learning




Supervised Learning

In supervised learning there are input varaibles (such as x,m,n) and output variables (think Y). An algorithm is used to try to understand the mapping function from the input to the output.If this is done successfully, new input variables (x,m,n..) can be entered into the algorithm, and have a high chance of predicting the output (Y)

Plain English: We feed the algorithm input (data) as well as desired output, and while processing the data the algorithm becomes 'smart' and able to make a connection between input and output. Think about a child flipping through a picture book with a teacher. The teacher knows the answer to what each animal is, and the child gives their best guess for what each animal is. If the child guesses incorrectly, the teacher can correct them and over time the child will hopefully learn

Typical Supervised learning problems

Classification: the output variable is categorical e.g. monkey, apple, bird, healthy, unhealthy, etc..

Regression: the output variable is a value e.g. 13, $340.41, 65%, 0, 1, etc..



Unsupervised Learning

Similar to supervised learning there are input varaibles (such as x,m,n), however there are no linked output variables This type of learning is used to better understand the shape and distribution of the data in order to learn more about it. With unsupervised, there is no teacher and no correct answer. The point of this learning to to let the algorithm run in an attempt to discover and display something interesting (usually valuable) about the data

Plain English: We feed the algorithm input (data) and hope it spits back something interesting that we might not have found on our own

Typical Unsupervised learning problems

Clustering: Find groups (clusters) within the data. That is, data that are more closely link to eachother than all other data points

Association: Find connecitons within the data. e.g. people who bought a large popcorn had a x% chance of buying soda as well.

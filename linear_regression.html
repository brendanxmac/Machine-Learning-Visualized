The Algorithm


  • Input:

    • K (# of clusters)
    • A data set

  • Setup: Randomly generate starting positions for the K many centriods


  • Step 1) Data Assignment: Assign each datapoint to the nearest centroid

  • Step 2) Centroid Update: For each centroid, calculate the mean distance of all data points assigned to it, update centroid location to mean location

  • Repeat steps 1 & 2 until no datapoint reassigned


In a Nutshell


Category

Unsupervised Learning- Data is not labeled

Usage:

Classification (Clustering) - Finding groups within the data

  • K Means can be thought of as clustering data into K groups.
  • After the variable K is defined, (for example purposes we will use 3) The data that is already present will separate itself into K groups.
  • Thinking of this algorithm visually, it appears as if the data actually "reveals" itself into K groups.
  • For all Clustering algorithms remember, they are searching and exposing data groups based on certain qualities within the data.
  • Over each run of the algorithm the groups become more and more defined.
